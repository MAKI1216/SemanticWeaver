# 2026-06-19 工作日志

## 音效系统 v3 全面重写
- **audio.js 完全重写** (v3)
  - 新增卷积混响基础设施（ConvolverNode + 4秒脉冲响应 + 压缩器）
  - 所有音效通过 sendToReverb() 发送到混响通路，增加空间感和温暖感
  - **关键词提取音 (playKeywordPickup)**：改为280Hz sine 水滴音，音量0.014，叠加高八度泛音，慢起音包络
  - **诊所氛围 (playClinicAmbience)**：Am-F-C-G 和弦进行（每和弦10秒，2秒淡入3秒淡出），sine pad，0.07Hz LFO呼吸感，每18秒高音shimmer，移除所有噪音层
  - **审讯室氛围 (playInterrogationAmbience)**：D1+A1+F2 深沉 sine drone + 锯齿波低通滤波扫描(0.03Hz LFO) + 每25秒三角波低音，大量混响，移除金属回响和噪音
  - **倒计时音乐 (startCountdownMusic)**：电影级弦乐 drone（锯齿波+低通滤波模拟弦乐），三层渐进：
    - Tier1: D2+A2 弦乐 + 1.2Hz 柔和脉冲
    - Tier2: 加 C#4 小二度不协和音 + 2Hz 脉冲
    - Tier3: 加 A#3 三全音 + D1 超低频 + E6 高频紧张泛音 + 3Hz 脉冲
  - **工业杂音→电影级低频冲击 (playIndustrialNoise)**：sine 80→35Hz 下滑 + 低通滤波 + 混响
  - **警报→弦乐紧张swell (playAlarmBurst)**：锯齿波 220→330Hz 渐强 + 低通滤波 + 混响
  - **警报循环 (startAlarmLoop)**：锯齿波 330Hz + 0.8Hz LFO 频率调制 + 低通滤波
  - 其他音效全部柔化：playDrop/playDropLight 改为低频 sine 水滴；playFail 加低通滤波；playGlitch 降低频率和音量；playCountdownTick 降到150Hz；playContradictionMark sawtooth→sine
